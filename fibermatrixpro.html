Fiber Matrix PRO

📘 Fiber Matrix Pro — User Guide

Overview

Fiber Matrix Pro is a field-ready splice documentation tool. Pin your location, record fiber optic splices between cable pairs, and export a clear CSV report for each location.

Core Workflow

1. Record New Location

Click the blue "📍 Record New Location" button. The app will attempt to capture your GPS coordinates. If GPS is unavailable or denied, you'll still create the location and can edit coordinates later.

2. Define Cable/Device Inventory

When creating your first splice, a modal will appear to define your cable/device inventory for that location. Enter items separated by commas or line breaks (cables, devices, fiber optic pigtails, optical splitters, etc.). You can edit this list anytime by clicking the purple "🔌 Cables/Devices" button.

3. Add Splices

Click "+ Add Splice" on any location card. Select cable/devices from the dropdowns and confirm. You can add as many splices as needed.

4. Edit & Manage

Click "✏️ Edit" to update location details, or "🔌 Cables/Devices" to manage your inventory. Each splice can be edited with the "✏️ Edit Splice" button.

5. Save & Export

Click "💾 Save" to download a JSON backup. Click "📊 Export Excel CSVs" to download a zip file with spreadsheet-ready exports.

Auto-Save

Your work is automatically saved every 30 seconds to your browser's local storage. If you close the tab and reopen it, your data will be restored automatically. Use the manual "💾 Save" button for portable JSON backups.

Cable/Device Management

Per-Location Inventory: Each location has its own cable/device inventory. Enter items when creating your first splice, or manage them later via the "🔌 Cables/Devices" button.

What to Include: Cables, devices, fiber optic pigtails, optical splitters, or any other network components at the location.

Input Format: Separate items by commas or place each on a new line:

48ct East, node tail, 8ch DMX
or
48ct East
node tail
8ch DMX
