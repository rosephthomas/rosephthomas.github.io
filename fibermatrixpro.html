Fiber Matrix PRO

📘 Fiber Matrix Pro — User Guide

Overview

Fiber Matrix Pro is a field-ready splice documentation tool. Pin your location, record fiber optic splices between cable pairs, and export a clear CSV report for each location.

Core Workflow

1. Record New Location

Click the blue "📍 Record New Location" button. The app will attempt to capture your GPS coordinates. If GPS is unavailable or denied, you'll still create the location and can edit coordinates later.

2. Add Splices

Click "+ Add Splice" on any location card. Fill in the splice details and confirm. You can add as many splices as needed.

3. Edit Location

Click "✏️ Edit" on a location to update its name and coordinates anytime.

4. Save Your Work

Click "💾 Save" to download a JSON backup file. This preserves all locations and splices for later reload.

5. Export CSVs

Click "📊 Export Excel CSVs" to generate one CSV per location. Each file includes the location name, coordinates, export date, and all splices in a spreadsheet-ready format.

Splice Fields Explained

Cable/Device ID 1 & 2: The identifiers for the two cables or devices being spliced (e.g., "CAB-001", "DEVICE-B").

Bundle (BNDL): The fiber bundle color (BLUE, ORANGE, GREEN, etc.). Can also use STRIPE variants (e.g., BLACK-STRIPE) for striped bundles.

Fiber (FIBR): The individual fiber color within the bundle.

Notes: Optional text field for splice notes (e.g., "Feeds Site C", "Temporary patch").

Tips

  • Always save your work frequently using the "💾 Save" button.
  • Use descriptive location names — they become CSV filenames.
  • The JSON backup is portable — load it on any device with this app.
  • Stripe colors (e.g., BLACK-STRIPE) are useful for identifying bundle sheath patterns.
  • Edit locations anytime to fix coordinates or names.
