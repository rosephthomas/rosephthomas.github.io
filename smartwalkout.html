Smart Walkout – Quick Guide

1. Add Points

  • Record Current GPS – enter a name and tap to capture your current phone GPS location.
  • You can edit a point’s name and coordinates directly in the table.

2. Connect Points

  • Use the canvas to visually connect points.
  • Tap one point, then another.
  • A dialog appears – optionally type a connection name (or leave blank) and tap Confirm.
  • The connection is drawn with the estimated distance in feet.

3. View & Edit Links

  • The Links table shows each connection, its name, endpoints, and distance.
  • You can rename links there or delete them.

4. Auto-Save

  • The indicator in the top-right shows Saved (green) or Unsaved changes (red).
  • Changes are auto-saved to your browser after a short delay.
  • When you reopen this page on the same device/browser, your last session reloads.

5. Export to KML

  • Tap Export to .kml to download a KML file.
  • Open the KML in Google Earth Pro to see:
    • Placemarks for each point (with your names) inside a “Stations” folder.
    • Lines for each connection, with distance in the description, inside a “Lines” folder.

6. Clear Session

  • Clear Session deletes all points, links, and saved data for this project from this browser.
  • Use this when you want to start a completely new project.
