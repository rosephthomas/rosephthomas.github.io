📡 How to Use
What This Does

Network Station Logger captures your GPS coordinates and mobile network data at each physical location you visit. Press Record Station at each point — it will acquire GPS and snapshot the current network conditions. Build up a list of stations, then export for analysis.

Buttons
ButtonWhat it does
⊕ Record StationAcquires GPS fix then captures all network data. Adds a new numbered station to the list.
⬇ CSVExports all stations as a comma-separated file. Opens in Excel, Google Sheets, etc.
🗺 KMLExports stations as a KML file. Open in Google Earth or Google Maps to view pins with metadata popups.
🗑 ClearDeletes all stations after confirmation. Cannot be undone.
✕ DeleteRemoves a single station from the list.

Notes

Network data is a browser estimate — values update when the OS detects a meaningful change, not on a fixed timer. Two nearby stations may share identical network readings if signal didn't change enough to trigger an update.

Network Information API requires a Chromium-based browser (Chrome, Edge). GPS requires location permission and HTTPS.

All data is auto-saved to your browser's local storage. It will survive page refreshes and closing the tab.
